# 📰 Identify Latent Topics in News Articles using LDA Topic Modeling

## 📌 Project Overview

This project focuses on identifying hidden (latent) topics in a collection of news articles using **Latent Dirichlet Allocation (LDA)**, a popular topic modeling technique in Natural Language Processing (NLP). The system automatically discovers meaningful topics from textual news data and classifies unseen news headlines into the most relevant topic category.

The project demonstrates the complete NLP pipeline including text preprocessing, topic extraction, topic interpretation, classification, and model evaluation using machine learning techniques.

---

## 🎯 Objectives

* Discover hidden topics from a collection of news articles.
* Perform text preprocessing and feature extraction.
* Apply Latent Dirichlet Allocation (LDA) for topic modeling.
* Assign meaningful labels to discovered topics.
* Classify unseen news headlines into relevant topics.
* Evaluate model performance using Random Forest Classification and Cross Validation.

## 📊 Dataset

The dataset consists of news articles collected from various categories. The textual data undergoes preprocessing before being used for topic extraction and classification.

### Data Preprocessing Steps

* Text Cleaning
* Tokenization
* Stop Word Removal
* Lemmatization
* Creation of Bag-of-Words Representation
* Dictionary Construction

---

## 🏗️ Methodology

### 1. Text Preprocessing

Raw news articles are cleaned and transformed into machine-readable representations.

### 2. Topic Modeling using LDA

Latent Dirichlet Allocation (LDA) is used to discover hidden thematic structures within the news corpus.

### 3. Topic Interpretation

The generated topics are analyzed and assigned meaningful labels such as:

* Health
* Finance
* Technology
* Social Media
* Entertainment
* Business
* Current Affairs

### 4. Headline Classification

New and unseen headlines are processed and assigned to the most relevant topic based on the topic distribution generated by the trained LDA model.

### 5. Performance Evaluation

A Random Forest classifier is used for classification and evaluated using:

* Training Accuracy
* Testing Accuracy
* Confusion Matrix
* Cross Validation (CV = 3)
* Cross Validation (CV = 5)
